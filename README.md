# 短链接系统

一个简洁、功能完善的短链接生成与管理系统，支持后台管理、人机验证和点击统计。

## ✨ 功能特性

- 🔗 **短链接生成** - 输入长链接，一键生成简洁短码
- ✏️ **自定义短码** - 支持自定义短码，打造专属链接
- 📊 **数据统计** - 实时追踪每个链接的点击次数和详细记录
- 🛡️ **人机验证** - 图形验证码防护，防止恶意刷量
- 🔐 **后台管理** - 完善的管理员后台，支持链接的查看、搜索、编辑、启禁用、删除
- 🔑 **密码修改** - 管理员可在后台修改登录密码
- 📱 **响应式设计** - 完美适配桌面和移动设备
- 🗄️ **SQLite 数据库** - 无需额外配置，开箱即用

## 🚀 安装部署

### 环境要求

- PHP >= 7.4
- PDO SQLite 扩展
- Apache（推荐）或 Nginx

### 安装步骤

1. **上传文件** - 将所有文件上传到 Web 服务器目录

2. **设置权限** - 确保 `data/` 目录可写：
   ```bash
   chmod 755 data/
   ```

3. **访问使用** - 打开浏览器访问项目目录即可

4. **配置 URL 重写**（可选）
   - **Apache**: `.htaccess` 已包含重写规则，确保开启了 `mod_rewrite`
   - **Nginx**: 在配置中添加：
     ```nginx
     location /shortlink/ {
         try_files $uri $uri/ /shortlink/go.php?c=$1;
     }
     location ~ ^/shortlink/([a-zA-Z0-9_-]+)$ {
         rewrite ^/shortlink/([a-zA-Z0-9_-]+)$ /shortlink/go.php?c=$1 last;
     }
     ```

## 📋 使用说明

### 生成短链接

1. 在首页输入目标网址
2. 输入验证码
3. 点击"生成短链接"
4. 复制生成的短链接即可分享

### 自定义短码

1. 点击"自定义短码"展开选项
2. 输入自定义短码（字母、数字、下划线、短横线，2~20位）
3. 提交生成

### 后台管理

1. 点击"管理员登录"进入后台
2. 输入管理员账号密码和验证码
3. 在仪表盘中可以：
   - 查看所有链接和统计数据
   - 搜索链接（按短码或网址）
   - 启用/禁用链接
   - 编辑目标网址
   - 查看点击详情
   - 删除链接
   - 修改管理员密码
